How to create Java project using gradle

  • Create gradle based directory structure along with gradle wrapper and gradle build file
    • gradle init --type java-library

  • Now edit build.gradle and add idea, eclipse and jetty plugin apply plugin: 'idea' apply plugin: 'eclipse' apply plugin: 'jetty'

  • Add .gitignore file
  • Add travis file
