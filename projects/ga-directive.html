ga Directive

AngularJS directive for Google Analytics

This directive is used in AngularJS application to track GA events. Source code available here

Design

This project contains two modules
  • analyticsModule in AnalyticsService.js file
  • gaDirectiveModule in GADirective.js file

analyticsModule

This angularjs module has google analytics service named AnalyticsServices, which has two methods

  • trackEvent
    • trackEvent method is uesd to track events within your project, for example you can track how many users clicked on a particular button, then you can use it to track butonClick events. To use trackEvent method you need to pass following parameters
    • eventCategory
    • eventAction
    • eventLabel
            AnalyticsServices.trackEvent("Gallery", "Edit", "Name");
  • trackPageView
    • trackPageView method is used to track page view. It needs only one parameter to be passed
    • url
            AnalyticsServices.trackPageView("/accounts");

gaDirectiveModule

gaDirectiveModule wraps AnalyticsServices in directive so that avialble methods can be used in HTML directly. There are three directives:

  • Tracking page views
    •         <div sw-track-page-view pageurl="/account" /div>
  • Tracking Events 
    •         <a sw-track-event eventcategory="My Account" eventaction="Upgrade" eventlabel="Custom Posts">Learn More </a>
  • Track Page views only once when page loads
    •         <div sw-track-page-view-on-load pageurl="/account" /div>
